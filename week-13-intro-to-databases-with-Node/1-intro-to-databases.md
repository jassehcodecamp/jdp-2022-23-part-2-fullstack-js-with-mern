## What is a Database

- A Database is a collection of data stored in a format that can be easily accessed.
- A Database is a repository/collection of a lot Data.


## A Database Management System (DBMS)

- We used a DBMS to manage our Databases.
- We have several DBMSes out there, and generally they are classified into two groups **Relational** and **Non-Relational** Database Management Systems.

- In this course we will be dealing with Non-Relational Databases, and we will be using one of the most popular out there called **MongoDB**

## MongoDB

- MongoDB is an NoSQL or document-based database systems.
